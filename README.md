# Mini-Designer

[![ci](https://github.com/ivan-zozulia/mini-designer/actions/workflows/ci.yml/badge.svg)](https://github.com/ivan-zozulia/mini-designer/actions/workflows/ci.yml)

A mini t-shirt designer.

## 🦄 Stack

- Nuxt 4
- Vue 3
- Pinia
- VueUse
- Tailwind CSS 4
- Vitest
- Playwright

## 📦 Local Setup

```bash
npm install
npm run dev
```

## 🧪 Tests

```bash
npm run test:run
```
